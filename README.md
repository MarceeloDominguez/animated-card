## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/animated-card.git
```

Go to the project directory

```bash
  cd animated-card
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
